# WebOS 项目总览

> 版本: 0.0.1beta | 日期: 2026-06-14

## 这是什么

WebOS 是一个运行在浏览器中的操作系统。不是模拟器，不是主题，是真正的操作系统——进程管理、内存管理、文件系统、音频服务、窗口管理，全都有。整个画面通过 WebGPU 渲染到一个全屏画布上，F12 看到的只有一个 `<canvas>`。

## 启动链

```
TypeScript Launcher → C Bootloader → C Kernel → Login System → Desktop Environment
```

TypeScript 只在最外层（入口）和最上层（桌面），核心 OS 全部用 C/C++ 实现，编译为 WASM。

## 目录结构

```
webos/
├── host/                    # TypeScript 启动器
├── bootloader/              # C 引导加载器 → bootloader.wex
├── kernel/                  # C 内核 → kernel.Wdll
├── drivers/                 # 设备驱动
├── services/                # 系统服务
├── libs/                    # 共享库
├── apps/                    # 应用程序
├── desktop/                 # 桌面环境 (TypeScript)
├── tools/                   # 构建工具
├── tests/                   # 测试
├── docs/                    # 文档
│   └── dev-plan/            # 开发规划文档
├── site/                    # 官方网站
├── Makefile                 # 顶层构建
└── README.md
```
