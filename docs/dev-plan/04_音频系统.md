# 音频系统

WebOS 的音频系统类似 PulseAudio——所有应用的音频先送到音频服务器混合，然后统一输出到浏览器。

---

## Stream 数据结构

```c
typedef struct {
    int stream_id;
    int sample_rate;      // 44100
    int channels;         // 1=mono, 2=stereo
    float volume;         // 0.0 ~ 1.0
    int is_paused;
    int owner_pid;
    float buffer[4096];   // 环形缓冲区 (frames × channels)
    int write_pos;
    int read_pos;
    int available;        // 可读帧数
} audio_stream_t;
```

- 最大 **16** 个同时活跃的音频流
- 缓冲区大小: **4096** 帧

---

## 混合流程

```
1. 收集所有未暂停 Stream 的可读数据
2. 逐帧叠加: mixed[i] += stream[j].data[i] * stream[j].volume
3. 应用主音量: mixed[i] *= master_volume
4. 软削波: tanh(mixed[i]) 防止爆音
5. 写入 Audio Driver 输出缓冲区
```

### 削波策略

使用 `tanh()` 软削波，比硬削波听感更自然:

```c
float soft_clip(float sample) {
    return tanhf(sample);  // 平滑过渡，不会突变
}
```

## audio_client API

应用通过这个库与音频服务器交互:

```c
// 打开/关闭音频流
int audio_open(int sample_rate, int channels);  // 返回 stream_id
int audio_close(int stream_id);

// 写入 PCM 数据
int audio_write(int stream_id, const void* data, int frames);

// 控制
int audio_set_volume(int stream_id, float volume);  // 0.0 ~ 1.0
int audio_pause(int stream_id);
int audio_resume(int stream_id);

// 便利函数
void audio_play_tone(int stream_id, float freq, float duration_sec);
void audio_play_noise(int stream_id, float duration_sec);
```

### PCM 格式

- 采样率: **44100 Hz**
- 位深: **32-bit float**（Web Audio API 原生格式）
- 声道: 单声道或立体声
- 帧大小: channels × sizeof(float)

## audio_driver 接口

驱动层直接对接浏览器 Web Audio API:

```c
// 初始化
int audio_driver_init(void);

// 写入输出缓冲区
int audio_driver_write(const float* data, int frames);

// 设置主音量
int audio_driver_set_volume(float volume);

// 获取当前播放位置
int audio_driver_get_position(void);
```

### JS 端实现

```javascript
// 在 RuntimeBridge 中:
js_audio_create_context: () => {
    const ctx = new AudioContext({ sampleRate: 44100 });
    // 存储在 audioContexts Map 中
    return ctxId;
}

js_audio_play_pcm: (dataPtr, len, sampleRate) => {
    const data = new Float32Array(memory.buffer, dataPtr, len);
    const buffer = audioCtx.createBuffer(2, len/2, sampleRate);
    buffer.getChannelData(0).set(data.filter((_, i) => i % 2 === 0));
    buffer.getChannelData(1).set(data.filter((_, i) => i % 2 !== 0));
    const source = audioCtx.createBufferSource();
    source.buffer = buffer;
    source.connect(audioCtx.destination);
    source.start();
}
```

## 音量控制

| 层级 | 控制 | 默认值 |
|------|------|--------|
| 主音量 | `audio_driver_set_volume()` | 0.80 |
| 流音量 | `audio_set_volume()` | 1.00 |
| 通知音量 | 通过 Settings 应用 | 0.70 |

混合时: `final = sample × stream_volume × master_volume`
