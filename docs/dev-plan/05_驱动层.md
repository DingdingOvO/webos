# 驱动层

WebOS 的驱动层封装浏览器 API，给内核和上层服务提供统一的设备接口。每个驱动编译为独立的 .Wdll 模块。

---

## 驱动架构

```
┌─────────────────────────────────┐
│         Kernel / Services        │
├─────────────────────────────────┤
│     Driver API (C interface)    │
├──────┬──────┬──────┬──────┬─────┤
│ GPU  │Input │Store │Net   │Audio│
│Driver│Driver│Driver│Driver│Drvr │
├──────┴──────┴──────┴──────┴─────┤
│    RuntimeBridge (JS host)      │
├─────────────────────────────────┤
│    Browser APIs (WebGPU, etc.)  │
└─────────────────────────────────┘
```

---

## GPU 驱动

封装 WebGPU API，提供基础 2D 渲染接口:

```c
// 上下文
int  gpu_init(int width, int height);
void gpu_shutdown(void);

// 绘制
void gpu_clear(float r, float g, float b, float a);
void gpu_draw_rect(int x, int y, int w, int h, float color[4]);
void gpu_draw_rounded_rect(int x, int y, int w, int h, int radius, float color[4]);
void gpu_draw_text(int x, int y, const char* text, float color[4], int font_size);
void gpu_draw_line(int x1, int y1, int x2, int y2, float color[4], int width);
void gpu_draw_circle(int cx, int cy, int radius, float color[4]);

// 帧缓冲
void gpu_present(void);
int  gpu_create_surface(int w, int h);   // 创建离屏渲染目标
void gpu_destroy_surface(int id);
void gpu_blit_surface(int id, int x, int y, int w, int h);

// 模糊
void gpu_blur_region(int x, int y, int w, int h, int radius);

// 噪点
void gpu_apply_noise(int x, int y, int w, int h, float opacity, float scale);
```

## 输入驱动

捕获键盘和鼠标事件，维护事件队列:

```c
// 事件类型
#define INPUT_EVENT_KEY_DOWN   1
#define INPUT_EVENT_KEY_UP     2
#define INPUT_EVENT_MOUSE_MOVE 3
#define INPUT_EVENT_MOUSE_DOWN 4
#define INPUT_EVENT_MOUSE_UP   5
#define INPUT_EVENT_SCROLL     6

typedef struct {
    int type;
    int keycode;       // 键码
    int modifiers;     // Ctrl/Shift/Alt/Meta
    int mouse_x;
    int mouse_y;
    int mouse_button;  // 0=left, 1=middle, 2=right
    int scroll_delta;
} input_event_t;

// API
int  input_poll_events(input_event_t* buf, int max_count);
int  input_wait_event(input_event_t* event, int timeout_ms);
int  input_get_mouse_x(void);
int  input_get_mouse_y(void);
int  input_get_key_state(int keycode);
```

- 事件队列最大 **128** 条
- JS 端监听 `keydown`, `keyup`, `mousemove`, `mousedown`, `mouseup`, `wheel`

## 存储驱动

封装 IndexedDB，提供块级读写接口（供 WebFS 使用）:

```c
// 块级操作
int  storage_read_block(uint32_t block_num, void* buf);
int  storage_write_block(uint32_t block_num, const void* buf);
int  storage_get_block_count(void);

// 高级文件操作（供 fs_service 使用）
int  storage_read_file(const char* path, void* buf, int* size);
int  storage_write_file(const char* path, const void* buf, int size);
int  storage_exists(const char* path);
int  storage_mkdir(const char* path);
int  storage_unlink(const char* path);
int  storage_listdir(const char* path, void* entries, int max);
```

## 网络驱动

封装 Fetch API:

```c
typedef struct {
    int status;          // HTTP 状态码
    char content_type[64];
    int content_length;
    void* body;          // 响应体
} http_response_t;

// HTTP 请求
int  net_http_get(const char* url, http_response_t* resp);
int  net_http_post(const char* url, const void* body, int body_len,
                   const char* content_type, http_response_t* resp);

// WebSocket
int  net_ws_connect(const char* url);
int  net_ws_send(int conn_id, const void* data, int len);
int  net_ws_recv(int conn_id, void* buf, int max_len);
int  net_ws_close(int conn_id);
```

- 最大 **16** 个并发网络连接

## 驱动注册表

所有驱动在加载时向内核注册:

```c
typedef struct {
    const char* name;
    uint32_t version;
    int (*init)(void);
    void (*shutdown)(void);
} driver_info_t;

int driver_register(const driver_info_t* info);
int driver_unregister(const char* name);
const driver_info_t* driver_get(const char* name);
```

注册表在内核初始化时创建，驱动通过 `EMSCRIPTEN_KEEPALIVE` 导出注册函数。
