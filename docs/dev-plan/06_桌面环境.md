# 桌面环境

WebOS 桌面环境完全用像素绘制——没有 DOM 元素，所有你看到的都是画布上的像素。

---

## 窗口管理器

### 窗口操作

| 操作 | 说明 |
|------|------|
| 创建 | `wm_create_window(title, x, y, w, h)` |
| 销毁 | `wm_destroy_window(handle)` |
| 移动 | 拖拽标题栏 |
| 缩放 | 8 个方向的边缘拖拽 |
| 最小化 | 标题栏按钮或任务栏点击 |
| 最大化 | 标题栏按钮或双击标题栏 |
| 关闭 | 标题栏按钮 |
| 聚焦 | 点击窗口任意位置 |

### Z 序

- 窗口按 Z 序从下到上绘制
- 点击窗口时移到最上层
- 最小化窗口不参与绘制

### 标题栏

- 高度: **36px**
- 左侧: 应用图标 + 窗口标题
- 右侧: 最小化(−), 最大化(□), 关闭(✕) 三个圆形按钮
- 聚焦窗口: 顶部 2px 蓝色辉光线
- 非聚焦窗口: 标题栏颜色变暗

### 缩放手柄

8 个方向的 6px 边缘热区: N, NE, E, SE, S, SW, W, NW

- 光标自动切换: `ns-resize`, `nesw-resize`, `ew-resize`, `nwse-resize`
- 最小窗口: 200×120

### 边缘吸附

窗口拖到屏幕边缘 10px 内自动贴合。

## 任务栏

- 位置: 屏幕底部，全宽
- 高度: **48px**
- 设计: 模糊背景 (blur 40px) + 噪点，透明度 0.78

### 三区布局

```
┌──────────┬──────────────────┬──────────────┐
│  左侧    │     中间          │    右侧       │
│ 开始按钮  │  运行中应用       │  系统托盘     │
│  ⊞ 48×48 │  [App] [App] ... │ 📶 🔊 🕐 14:30│
└──────────┴──────────────────┴──────────────┘
```

| 区域 | 内容 | 宽度 |
|------|------|------|
| 左侧 | 开始按钮 | 48×48px |
| 中间 | 运行中应用的图标 | 弹性 |
| 右侧 | WiFi + 音量 + 电池 + 时钟 | 固定 ~200px |

### 系统托盘

- WiFi 图标
- 音量图标
- 电池图标（如果可用）
- 时钟（实时更新）

## 开始菜单

- 触发: 点击开始按钮
- 位置: 任务栏左上方
- 尺寸: 300×400px
- 圆角: 16px（仅上方）
- 设计: blur(32px) + 噪点，透明度 0.85

### 内容

1. **搜索栏** — 顶部，圆角 18px，输入框样式
2. **置顶应用** — 3×N 网格，图标 + 名称
3. **最近使用** — 列表
4. **电源** — 底部：关机、重启、睡眠

## 通知系统

- 位置: 屏幕右上角
- 宽度: 320px
- 圆角: 10px
- 自动消失: 5 秒
- 动画: 右侧滑入 (300ms, cubic-bezier 弹性)

### 通知类型

| 类型 | 左侧色条 |
|------|----------|
| info | 蓝色 |
| success | 绿色 |
| warning | 黄色 |
| error | 红色 |

### 通知内容

- 图标
- 标题 (bold)
- 时间戳
- 正文

## 右键菜单

- 触发: 右键点击桌面 / 图标 / 窗口
- 圆角: 8px
- 设计: blur(32px) + 噪点
- 菜单项高度: 32px
- 分割线: 1px border 颜色

### 桌面右键菜单

- 刷新桌面
- 更换壁纸
- 新建文件夹
- 排列图标

## 壁纸

默认壁纸: 深蓝色渐变 `#0F172A → #1E293B → #0F172A`

在 WebGPU 中用程序化生成:

```
1. 创建全尺寸纹理 (width × height × rgba8unorm)
2. 逐像素写入渐变色
3. 上传到 GPU
4. 壁纸作为窗口/任务栏模糊的源纹理
```

支持自定义壁纸（通过 Settings 应用切换），存储在 `/home/user/wallpapers/`。
