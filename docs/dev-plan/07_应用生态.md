## 应用详情

### 1. Terminal
命令行终端模拟器，内置 shell。支持命令历史、Tab 补全、行编辑。内置命令: help, clear, echo, pwd, cd, ls, mkdir, cat, rm, ps, uname, date, whoami, exit。

### 2. File Manager
图形化文件管理器。侧边栏收藏夹（Home, Desktop, Documents, Downloads 等），主区域网格/列表视图切换，右键菜单操作（复制、粘贴、删除、重命名）。

### 3. Text Editor
轻量文本编辑器。多行编辑、行号、基本语法高亮（C/JSON）、查找替换、Tab 缩进、Ctrl+S 保存。

### 4. Settings
系统设置。分类: Display（壁纸/亮度/分辨率）、Sound（音量/通知音）、Network（WiFi/代理）、Personalization（主题/强调色/噪点）、About（系统信息）。

### 5. Calculator
简单计算器。4×4 按钮网格，支持四则运算，小数点，清除，等号。

### 6. Paint
简单画板。画笔工具、8 色调色板、鼠标绘制、笔刷大小调节。

### 7. Music Player
音乐播放器。播放/暂停/上一首/下一首、进度条、音量控制、频谱可视化、播放列表管理、循环/随机模式。

### 8. System Monitor
系统监视器。CPU/内存使用率实时图表（60 秒历史）、进程列表（可排序/可杀进程）、文件系统使用量、音频流状态。

### 9. Browser
简易浏览器。URL 栏、导航按钮、使用网络服务获取页面内容。

### 10. App Store
应用商店。浏览可用应用、安装/卸载、应用详情、依赖检查。

## 应用开发规范

### 1. app.json 清单

每个应用必须有 `app.json`:

```json
{
  "name": "My App",
  "id": "com.example.myapp",
  "version": "0.0.1beta",
  "description": "App description",
  "entry": "myapp.wex",
  "icon": "/icons/myapp.png",
  "category": "Utilities",
  "author": "Author Name",
  "permissions": ["wm", "input", "fs"],
  "dependencies": []
}
```

### 2. 入口函数

```cpp
int main(int argc, char** argv) {
    int win = wm_create_window("My App", 100, 100, 640, 480);
    wm_show_window(win);
    emscripten_set_interval(tick, 16, NULL);  // 60fps 主循环
    return 0;
}
```

### 3. 渲染循环

所有渲染在 tick 回调中完成:

```cpp
void tick(void* arg) {
    // 1. 处理输入
    // 2. 更新状态
    // 3. 渲染到窗口表面
    // 4. GPU present
}
```

### 4. 权限声明

应用必须在 app.json 中声明需要的权限，未声明的 API 调用会返回错误。

### 5. 资源释放

应用退出前必须:
- 销毁所有窗口
- 关闭所有文件描述符
- 关闭音频流
- 释放共享内存

### 6. 错误处理

所有 API 调用检查返回值，负数表示错误:

| 错误码 | 值 | 说明 |
|--------|-----|------|
| ERR_INVAL | -1 | 无效参数 |
| ERR_NOENT | -2 | 不存在 |
| ERR_NOMEM | -3 | 内存不足 |
| ERR_BUSY | -4 | 资源忙 |
| ERR_PERM | -5 | 权限不足 |
| ERR_FAULT | -6 | 段错误 |
| ERR_EXIST | -7 | 已存在 |
| ERR_NOSYS | -8 | 系统调用未实现 |

### 7. 设计语言遵循

所有应用 UI 必须遵循 Frosted Glass 设计语言（见 01_设计语言.md）:
- 使用 Accent Blue (#3B82F6) 作为强调色
- 窗口背景使用 blur + 半透明 + 噪点
- 按钮圆角 6px，卡片圆角 10px
- 完整的四态交互（Default/Hover/Active/Focused）
